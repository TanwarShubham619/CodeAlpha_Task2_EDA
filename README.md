## Dataset Overview
The dataset used for this task contains quotes, authors, and tags scraped from a public website. The dataset was collected using Octoparse during Task 1.

- File Name: quotes.csv
- Total Records: 100
- Total Columns: 12
- Data Type: Text data

## Objectives
- Understand the dataset structure
- Identify patterns and trends
- Ask meaningful analytical questions
- Detect data quality issues

## Questions Asked
1. Which author appears most frequently?
2. What are the most common tags?
3. How many unique authors are present?
4. Are specific authors linked to particular themes?
5. What trends can be observed from tag distribution?

## Analysis & Observations
- Some authors contribute multiple quotes.
- Common tags include inspirational, life, love, and books.
- The dataset mainly reflects motivational and philosophical themes.
- Tag information is distributed across multiple columns.

## Data Issues Identified
- Multiple tag columns require consolidation for advanced analysis.
- No missing values were observed.

## Conclusion
Exploratory Data Analysis helped in understanding the dataset, identifying key patterns, and highlighting areas for further data cleaning and visualization.
